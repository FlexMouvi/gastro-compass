# 🧭 Gastro COMPASS — веб-платформа для гастрономического туризма

**Gastro COMPASS** — это веб-приложение, которое помогает пользователям находить и бронировать гастрономические туры по Красноярску, изучать рестораны, оставлять отзывы и планировать свои путешествия.

---

## ✨ Функциональность

### 👥 Для всех посетителей
- 📋 Просмотр списка гастрономических туров с краткой информацией (фото, название, локация, цена, рейтинг).
- 🔍 Детальная страница каждого тура с описанием, продолжительностью, списком включённых ресторанов.
- 🍽️ Страницы ресторанов с меню и отзывами.
- 📝 Регистрация и вход в личный кабинет.

### 🔐 Для авторизованных пользователей
- ⭐ Возможность оставлять отзывы на туры и рестораны (оценка по 5-балльной шкале, комментарий).
- 📅 Бронирование туров через модальное окно (имя, телефон, email, желаемая дата, количество гостей, комментарий).
- 🤖 Автоматическое заполнение формы бронирования данными из профиля.

### 🛠️ Для администратора
- 🧰 Полноценная административная панель для управления:
  - 🏨 ресторанами и блюдами
  - 🗺️ турами и включёнными в них ресторанами
  - 💬 отзывами на туры и рестораны
  - 📦 бронированиями
  - 👤 пользователями и их профилями
- 📊 Просмотр и редактирование любых записей.

---

## 🧰 Технологии

- **🐍 Бэкенд:** Python 3.14, Django 6.0.3
- **🗄️ База данных:** PostgreSQL (реляционная БД, данные сохраняются между запусками)
- **🎨 Фронтенд:** HTML5, CSS3, Bootstrap 5, JavaScript
- **📚 Библиотеки:** Pillow (обработка изображений), psycopg2-binary (драйвер PostgreSQL)
